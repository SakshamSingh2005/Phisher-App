🚨 Phisher App — Cybersecurity Awareness Project
📖 Description

The Phisher App is a cybersecurity-focused project designed to demonstrate how phishing attacks are structured and executed in a controlled, educational environment.

This project aims to raise awareness about phishing techniques and help users understand how to identify and prevent such attacks.

✨ Key Features
🔐 Simulated phishing login interface
📊 Demonstration of credential capture (for learning purposes)
🧠 Educational insight into social engineering attacks
⚡ Lightweight and easy-to-understand implementation
🛠️ Tech Stack
Frontend: Flutter,Dart
Backend: Node.js
Version Control: Git & GitHub

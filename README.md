🚨 Phisher App 
📌 Overview

This project is a Phisher App developed for educational and cybersecurity awareness purposes.
It demonstrates how phishing attacks work and helps users understand how such attacks can be identified and prevented.

⚙️ Features
Simulated login pages
Data capture demonstration (for learning only)
Simple UI for understanding phishing workflow
Beginner-friendly implementation
